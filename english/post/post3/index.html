Giới Thiệu Về Java Và Lập Trình Hướng Đối Tượng

Java là một ngôn ngữ lập trình mạnh mẽ, phổ biến trong nhiều lĩnh vực phát triển phần mềm, đặc biệt là các ứng dụng doanh nghiệp và web. Bài viết này sẽ giới thiệu về Java và lập trình hướng đối tượng (OOP) – một trong những yếu tố cốt lõi của Java.

Khái niệm về Lập trình Hướng Đối Tượng

Lập trình hướng đối tượng (OOP) là một phương pháp lập trình trong đó mọi thứ được coi là đối tượng, với các thuộc tính và phương thức. Java là một ngôn ngữ hoàn toàn hướng đối tượng, với bốn đặc điểm chính:

  • Encapsulation (Đóng gói): Dữ liệu và phương thức được nhóm lại trong một đối tượng.
  • Abstraction (Trừu tượng): Giấu đi chi tiết cài đặt, chỉ cung cấp các giao diện cần thiết.
  • Inheritance (Kế thừa): Các lớp có thể kế thừa các thuộc tính và phương thức từ lớp khác.
  • Polymorphism (Đa hình): Khả năng của đối tượng có thể thay đổi hành vi tùy theo ngữ cảnh.

Ví dụ về OOP trong Java

class Animal {
    void sound() {
        System.out.println("Animal makes a sound");
    }
}

class Dog extends Animal {
    void sound() {
        System.out.println("Dog barks");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal animal = new Animal();
        animal.sound(); // Animal makes a sound

        Dog dog = new Dog();
        dog.sound(); // Dog barks
    }
}
