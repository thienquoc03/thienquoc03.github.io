Tổng quan về JavaScript

JavaScript là ngôn ngữ lập trình phổ biến được thiết kế để làm cho các trang web trở nên tương tác và sống động. Mặc dù xuất phát điểm chỉ là một công cụ nhỏ trong trình duyệt, ngày nay JavaScript đã trở thành một nền tảng phát triển toàn diện, từ giao diện người dùng đến xử lý phía máy chủ.

Một số cột mốc quan trọng

  • 1995: Brendan Eich tạo ra JavaScript chỉ trong 10 ngày tại Netscape. Ban đầu nó được gọi là “Mocha”, sau đó đổi thành “LiveScript” và cuối cùng được đặt tên là “JavaScript” nhằm tận dụng sự phổ biến của Java thời điểm đó.
  • 2009: Node.js ra đời, giúp JavaScript có thể chạy trên server, mở ra kỷ nguyên phát triển full-stack bằng một ngôn ngữ duy nhất.
  • Hiện nay: JavaScript là cốt lõi của công nghệ web hiện đại, sử dụng rộng rãi trong các framework như React, Angular, và Vue.

Đặc điểm nổi bật của JavaScript

  • Linh hoạt: Có thể sử dụng cho cả front-end và back-end, từ web đến ứng dụng di động và thậm chí IoT.
  • Dễ học: JavaScript có cú pháp đơn giản, dễ tiếp cận đối với người mới học lập trình.
  • Event-driven (Hướng sự kiện): Cho phép xử lý các sự kiện như click chuột, nhập liệu hoặc tải trang.
  • Non-blocking I/O: Nhờ mô hình bất đồng bộ, JavaScript xử lý tác vụ nhanh hơn, đặc biệt là trong các ứng dụng thời gian thực.
  • Hỗ trợ cộng đồng mạnh mẽ: Hệ sinh thái thư viện phong phú như npm giúp dễ dàng tích hợp các tính năng.

Các ứng dụng chính

  1. Giao diện người dùng (Front-end):
    Các thư viện và framework như React, Vue.js, và Angular giúp phát triển giao diện web phức tạp một cách dễ dàng.

  2. Máy chủ (Back-end):
    Node.js mở rộng khả năng của JavaScript để xử lý các tác vụ máy chủ, bao gồm API, cơ sở dữ liệu, và hơn thế nữa.

  3. Ứng dụng di động:
    Với React Native hoặc Ionic, bạn có thể xây dựng ứng dụng di động đa nền tảng bằng JavaScript.

  4. Trò chơi:
    JavaScript, kết hợp với WebGL và các công cụ như Three.js, cho phép tạo ra các trò chơi 2D và 3D chạy trên trình duyệt.

Cú pháp minh họa

// Khai báo biến (sử dụng let hoặc const)  
const greeting = "Chào mừng bạn đến với JavaScript!";  

// Hàm cơ bản  
const square = (num) => num * num;  
console.log(square(5)); // Kết quả: 25  

// Điều kiện  
const age = 18;  
const message = age >= 18 ? "Đủ tuổi trưởng thành" : "Chưa đủ tuổi trưởng thành";  
console.log(message);  

// Vòng lặp  
const numbers = [1, 2, 3, 4, 5];  
numbers.forEach((num) => console.log(num));  

// Xử lý sự kiện (giả lập trong Node.js hoặc trình duyệt)  
document.addEventListener("click", () => {  
  console.log("Bạn vừa nhấp chuột!");  
});
